1️⃣ What is the difference between var, let, and const?
JavaScript Variable Types:

var: Function-scoped, allows re-declaration and reassignment
let: Block-scoped, allows reassignment but not re-declaration
const: Block-scoped, does not allow reassignment or re-declaration

2️⃣ What is the spread operator (...)?
Used to expand elements of arrays/objects for copying, merging, and updating data.

3️⃣ What is the difference between map(), filter(), and forEach()?
map() and filter() return new arrays, while forEach() is used for iteration without returning a value

4️⃣ What is an arrow function?
Arrow function is a shorter way to write functions in JavaScript using =>.

5️⃣ What are template literals?
Template literals are a way to write strings using backticks `, allowing variables and expressions inside ${}.
